# Panduan Pengguna (User Guide) - Modul HR

## Pendahuluan
Modul HR Bebang Sistem Informasi dirancang untuk memudahkan pengelolaan data karyawan, struktur organisasi, dan proses administrasi HR lainnya.

## Login
1. Akses aplikasi di `http://localhost:5173`.
2. Masukkan NIK dan Password Anda.
3. Klik tombol **Login**.

## Dashboard HR
Setelah login sebagai Staff HR, Anda akan melihat Dashboard yang berisi:
- Statistik total karyawan aktif.
- Daftar karyawan terbaru.
- Ringkasan cuti/izin (jika ada).

## Manajemen Karyawan
### Menambah Karyawan Baru
1. Buka menu **Data Karyawan**.
2. Klik tombol **+ Tambah Karyawan**.
3. Isi formulir data diri (Nama, NIK, No HP, dll).
4. Lengkapi data pekerjaan (Divisi, Jabatan, Tanggal Masuk).
5. Klik **Simpan**.

### Import Karyawan (Excel)
1. Buka menu **Import Karyawan**.
2. Unduh **Template Excel** yang disediakan.
3. Isi data karyawan pada file Excel sesuai format.
4. Unggah kembali file Excel tersebut.
5. Review data import, lalu konfirmasi.

### Edit Data Karyawan
1. Di daftar karyawan, klik tombol **Detail** atau **Edit** pada baris karyawan.
2. Ubah data yang diperlukan.
3. Anda juga dapat mengelola data **Keluarga** (Anak, Pasangan, Saudara) di tab "Keluarga".
4. Klik **Simpan Perubahan**.

## Pengajuan Pengunduran Diri (Resign)
### Mengajukan Resign
1. Buka menu **Pengunduran Diri**.
2. Klik **Buat Pengajuan**.
3. Pilih tanggal efektif resign dan alasan.
4. Submit pengajuan.

### Status Pengunduran Diri
Anda dapat memantau status pengajuan (Pending, Disetujui, Ditolak) pada halaman yang sama.
