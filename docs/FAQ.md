# FAQ & Troubleshooting

## Masalah Login
**Q: Saya tidak bisa login, muncul pesan "Invalid credentials".**
A: Periksa kembali NIK dan Password Anda. Jika lupa, hubungi Administrator untuk reset password.

**Q: Akun saya terkunci.**
A: Hubungi Admin untuk memastikan status akun Anda masih **AKTIF**.

## Masalah Import Excel
**Q: Import gagal dengan pesan "Invalid format".**
A: Pastikan Anda menggunakan **Template Excel Terbaru** yang diunduh dari sistem. Jangan mengubah header kolom.

**Q: Data tanggal lahir salah setelah import.**
A: Pastikan format tanggal di Excel adalah `YYYY-MM-DD` atau format Date Excel standar.

## Masalah Teknis
**Q: Website tidak bisa dibuka (Connection Refused).**
A:
1. Pastikan Anda terhubung ke jaringan kantor (jika intranet).
2. Laporkan ke tim IT jika server down.

**Q: Data tidak muncul di Dashboard.**
A: Coba refresh halaman (F5). Jika masalah berlanjut, kemungkinan ada gangguan koneksi ke Backend API.
